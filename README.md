## 진짜 수정한 점
- 단과대별 퀘스트 제거 → 통합 AI 퀘스트로 변경
- 학교 정보 탭이 빈칸으로 보이지 않도록 기본 데이터를 HTML에 내장
- 식단: 2026.06.13 토요일 공식 식단 페이지가 등록된 식단 없음인 점을 그대로 표시

- 학사일정: 기말고사, 성적입력, 여름계절수업 등 실제 일정 표시
- 부산대 마크를 크게 중앙 배치
- 산지니 공식 이미지 URL + 이미지 fallback 적용
- 발표용 10초 테스트 버튼 추가
- QR은 base64 이미지로 HTML 내부 포함
